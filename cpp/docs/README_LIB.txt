Building Sina as a library
==========================

If you're using this tarball to build Sina as a library, documentation on performing the library build can be found at docs/building.dox.


LC Documentation
----------------

If you're on LC, Sina's C++ documentation is available at https://lc.llnl.gov/workflow/docs/sina/cpp/index.html (as well as on the rz). The documentation includes tutorials for using Sina, as well as instructions for building it as a library (see "Building" in the navbar). If you have any questions, please email siboka@llnl.gov.
