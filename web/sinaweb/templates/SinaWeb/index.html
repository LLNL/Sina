SinaWeb

The SinaWeb is a demo application for visualizing the simulation management SQL data model. It's intended to provide context for the various workflows it can support, as well as get feedback for improvements. The schema is designed to be general for more control over the flow of data.

This demo does not represent a usable application and is not designed to provide specific functionality.
